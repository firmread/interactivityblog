OBJECT : Joker Heel

Joker Heel

Love & Hate Relationship

Collaborated with Shaan Jamal Sarang

For our project we decided to look at an object that people need to use but hate to use it and transform it into an object that will make the person laugh while using it | object that will make the person literally ‘laugh to death’ (a weapon for the Joker to use). The object we decided to look at were high heels because even though so many women complain about the pain they feel while wearing them, the very same women continue to buy and wear them.
